# Truth Boundary
The system may optimize for ATS and recruiter fit, but it must not create materially false claims.

## Allowed
- stronger wording for existing evidence
- clearer summary positioning
- surfacing adjacent transferable experience
- adding under-emphasized grounded skills

## Not Allowed
- fabricating direct experience with an exact cloud/platform
- converting AWS work into GCP work
- inventing production use of tools not supported by the selected track
- adding experience to roles that the selected track does not support
