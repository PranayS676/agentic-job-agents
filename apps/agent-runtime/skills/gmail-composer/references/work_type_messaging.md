# Work-Type Messaging

Use role-specific work-type wording.

## When the posting mentions contract, project, or C2C

1. It is acceptable to mention contract/project delivery fit.
2. It is acceptable to ask whether C2C is workable, but do it as a follow-up question, not a demand.
3. Example phrasing:
   - "My recent work has been strongly delivery-oriented in contract/project settings."
   - "If the profile is relevant, I would also be glad to confirm whether a C2C structure is workable."

## When the posting clearly mentions W2

1. Do not force "C2C only" into the email.
2. Keep the message focused on technical fit and recent delivery evidence.
3. If work-type discussion is necessary, keep it light and defer it to follow-up.

## When work type is unclear

1. Default to technical fit.
2. Do not add speculative contract/C2C language unless the job summary hints at it.
