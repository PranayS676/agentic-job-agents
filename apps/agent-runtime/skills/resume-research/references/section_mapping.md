# Section Mapping
Research output must only target these editable zones:
- `summary`
- `skills`
- one selected `experience_*` section

## Experience Target Rule
Use the most relevant recent role from the selected resume track.
Do not spread experience edits across multiple old roles.

## Remove Item Rule
If de-emphasis is needed, keep it inside the same constrained scope whenever possible.
Do not ask for broad resume surgery in research.
