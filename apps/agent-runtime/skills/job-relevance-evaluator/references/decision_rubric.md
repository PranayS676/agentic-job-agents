# Decision Rubric

Use a 0 to 10 relevance score.

1. Reward strong alignment with AI / ML / LLM / Python-heavy roles.
2. Reward C2C, project-based, and W2 opportunities.
3. Keep broad role coverage when the candidate profile clearly matches.
4. Penalize irrelevant roles, vague spam, and low-information recruiter messages.
5. Do not reject a strong-fit role solely because it is project-based, W2, or full-time.
