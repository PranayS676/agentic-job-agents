# Good Message Examples

## Fit example
Hi, I saw your ML Engineer role at Acme. My recent work includes Python-based AI delivery and production backend implementation that maps well to this scope. I have attached my resume here. If this looks relevant, happy to discuss next steps.

## Okayish example
Hi, I saw the Cloud Data Engineer opening at Acme. There is some overlap with my recent Python, data, and platform work, so I am sharing my resume for a quick review. If the fit looks close enough, we can discuss further.
