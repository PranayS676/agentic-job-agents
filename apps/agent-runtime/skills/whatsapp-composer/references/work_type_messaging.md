# Work-Type Messaging

## Contract / Project / C2C hints

1. It is acceptable to mention contract or project delivery fit.
2. It is acceptable to ask whether C2C is workable, but only as a light follow-up.
3. Keep technical fit first.

## W2 hints

1. Do not force C2C-only wording into the message.
2. Keep the message centered on role fit and attached resume.
3. If work-type discussion is needed, defer it until after initial interest.

## Unclear work type

1. Default to technical fit.
2. Do not speculate about contract structure.
