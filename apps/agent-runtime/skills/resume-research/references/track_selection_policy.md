# Track Selection Policy
Choose the resume track that gives the strongest truthful evidence for the target job.

## Primary Factors
1. Direct role-family match
2. Exact stack overlap
3. Relevant recent experience density
4. Explicit AI/ML/data/cloud evidence
5. Fewer hard gaps

## Exact Cloud Policy
If the JD requires GCP or Azure and the track only shows AWS, that is a hard gap.
Do not rewrite AWS into GCP or Azure experience.

## Okayish Roles
If the role is only okayish, still pick the best truthful track, but keep the edit plan narrower.
