# Good Email Examples

## Fit example
Subject: Senior Python Engineer - Pranay

Hi,

Your Senior Python Engineer opening aligns closely with my recent work on Python-first backend and AI delivery. In my recent work, I shipped production services using Python-based APIs and applied ML-oriented workflows that map well to the role requirements.

I have attached my tailored resume. If this looks relevant, I am available for a short call this week.

Subject: Cloud Data Engineer - Pranay

I saw the Cloud Data Engineer role and there is meaningful overlap with my recent Python, data, and platform-oriented delivery work, although the fit is narrower than a direct AI/ML role.

I have attached my resume for review. If the background looks close enough, I would be glad to discuss whether it is worth a short follow-up call.
